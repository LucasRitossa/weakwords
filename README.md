# weakwords <img src="extension/icons/icon128.png" width="40" height="40" alt="" />

Track your slowest and most errored words on MonkeyType.

## Install (Firefox)
Install the extension from [here]().

## Install (Chrome)
1. Download latest release.
2. Unzip the file
3. Type `chrome://extensions` into the URL bar.
4. Check "Developer mode"
5. Click "Load Unpacked" > Select unzipped directory

## Using Your Word Lists

1. In the popup, click copy all for slow or errored words
2. On monkeytype.com choose custom>change in the mode list
3. Paste into the text area, set words/time

## Features

- Tracks words as you type (slow by WPM, errors when you mistype)
- Settings: how many words to show, min samples for slow list, export/import, clear data
- Theme support

## Building

```bash
npm run pack
```
